# 💻 CSC-521 Object-Oriented Programming – Lab 3

This lab focuses on understanding and implementing **Encapsulation** in C++. Students will learn how to protect data members, provide controlled access through getters and setters, and apply validation logic to ensure secure and maintainable code.

---

# ⭐ Task: Product Class with Encapsulation
## 🔍 Overview

This task requires creating a `Product` class with proper encapsulation. The class should have private data members for name, price, and quantity, and public methods to set and get these values with validation.

## 🎯 Objective

To practice encapsulation by hiding internal data and exposing only safe, validated access through public methods.

## ✔️ Output

- Displays error messages when invalid values are set.  
- Shows correct product details when valid inputs are provided.

## ✍️ Learning Outcomes

- Define private data members to protect internal state  
- Implement public getters and setters with validation  
- Apply encapsulation principles to real-world scenarios
