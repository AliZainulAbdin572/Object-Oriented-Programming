# 💻 CSC-521 Object-Oriented Programming – Lab 05

This lab focuses on understanding and implementing different **types of inheritance** in C++. Students will explore single, multilevel, hierarchical, and multiple inheritance through practical tasks. The goal is to understand how classes can reuse and extend functionality across different inheritance structures.

---

# ⭐ Task 1: Single Inheritance
## 🔍 Overview

This task requires creating a `Person` class and a `Student` class that inherits from it. The `Student` class extends the functionality of `Person` by adding a student ID.

## 🎯 Objective

To practice single inheritance by deriving one class from a single base class.

## ✔️ Output

Displays complete student information using both base and derived class functions.

## ✍️ Learning Outcomes

- Implement single inheritance  
- Reuse attributes and functions from a base class  
- Extend functionality in a derived class  

---

# ⭐ Task 2: Multilevel Inheritance
## 🔍 Overview

This task involves designing a payroll system with three classes: `Person`, `Employee`, and `Manager`. Each class adds new attributes and functions, forming a parent-child-grandchild hierarchy.

## 🎯 Objective

To understand multilevel inheritance and how functionality can be extended across multiple levels.

## ✔️ Output

Displays complete manager information using all display functions from `Person`, `Employee`, and `Manager`.

## ✍️ Learning Outcomes

- Implement multilevel inheritance  
- Reuse and extend functionality across multiple levels  
- Apply inheritance to real-world organizational structures  

---

# ⭐ Task 3: Hierarchical Inheritance
## 🔍 Overview

This task requires creating an `Employee` base class and two derived classes: `Developer` and `Designer`. Each derived class adds its own unique attributes and functions.

## 🎯 Objective

To practice hierarchical inheritance where multiple classes inherit from a single base class.

## ✔️ Output

Displays information for both developer and designer objects, including base class attributes.

## ✍️ Learning Outcomes

- Implement hierarchical inheritance  
- Create multiple derived classes from one base class  
- Apply inheritance to diverse roles in an organization  

---

# ⭐ Task 4: Multiple Inheritance
## 🔍 Overview

This task involves building a `Photocopier` class that inherits from both `Printer` and `Scanner`. The `Photocopier` class combines functionalities of both base classes.

## 🎯 Objective

To understand multiple inheritance and how a class can combine features from more than one base class.

## ✔️ Output

Displays results of printing, scanning, and photocopying using the combined functionality.

## ✍️ Learning Outcomes

- Implement multiple inheritance  
- Combine features from multiple base classes  
- Apply inheritance to build multifunctional devices
