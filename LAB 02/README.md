# 💻 CSC-521 – Object-Oriented Programming – Lab 2

This lab focuses on understanding and implementing the fundamental concepts of Object-Oriented Programming (OOP) in C++ through five practical tasks. Students will explore class creation, object usage, member functions (inside and outside class), encapsulation, and problem-solving using OOP principles.

---

# ⭐ Task 1: Student Record System
## 🔍 Overview

This program creates a `Student` class with attributes for name, roll number, and marks. Member functions are defined inside the class to input and display student details.

## 🎯 Objective

To practice defining attributes and member functions inside a class and using objects to store and display data.

## ✔️ Output

Displays student information entered by the user.

## ✍️ Learning Outcomes

- Define classes with attributes  
- Implement member functions inside a class  
- Create and use objects to manage data  

---

# ⭐ Task 2: Rectangle (Functions Outside Class)
## 🔍 Overview

This program defines a `Rectangle` class with attributes for length and width. Member functions for setting dimensions, calculating area, and perimeter are defined outside the class using the scope resolution operator.

## 🎯 Objective

To understand how to declare functions inside a class and define them outside using scope resolution.

## ✔️ Output

Displays the area and perimeter of the rectangle.

## ✍️ Learning Outcomes

- Use scope resolution operator (::)  
- Define functions outside the class  
- Apply OOP concepts to geometric problems  

---

# ⭐ Task 3: Voting Eligibility
## 🔍 Overview

This program creates a `Voter` class with attributes for name and age. A function `isEligible()` checks if the voter is eligible (age > 18) and returns a boolean value.

## 🎯 Objective

To apply conditional logic within class functions and return values for decision-making.

## ✔️ Output

Displays whether the voter is eligible or not based on age.

## ✍️ Learning Outcomes

- Implement conditional checks in class methods  
- Return values from functions  
- Apply OOP to real-world scenarios  

---

# ⭐ Task 4: Temperature Converter
## 🔍 Overview

This program defines a `Temperature` class that converts Celsius to Fahrenheit using the formula:



\[
Fahrenheit = (Celsius * 9/5) + 32
\]



## 🎯 Objective

To implement mathematical formulas within class functions and display results.

## ✔️ Output

Displays the converted temperature in Fahrenheit.

## ✍️ Learning Outcomes

- Apply mathematical operations in class functions  
- Convert values between units  
- Reinforce encapsulation in problem-solving  

---

# ⭐ Task 5: Calculator
## 🔍 Overview

This program creates a `Calculator` class with attributes for two numbers. Functions for addition, subtraction, multiplication, and division are defined to perform operations based on user input.

## 🎯 Objective

To implement multiple operations within a single class and allow user interaction.

## ✔️ Output

Displays the result of the selected arithmetic operation.

## ✍️ Learning Outcomes

- Implement multiple functions in a class  
- Handle user input for operations  
- Apply OOP concepts to build a utility program  

---

# 🏠 Separate Header Files
## 🔍 Overview

All programs are reorganized using separate header files to improve structure and readability. Each class is placed in its own header file and included in the main program.

## 🎯 Objective

To practice modular programming and improve code organization.

## ✔️ Output

Programs run with improved structure using header files.

## ✍️ Learning Outcomes

- Create and use header files  
- Improve modularity and readability  
- Apply best practices in C++ programming
